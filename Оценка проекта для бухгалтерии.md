# Оценка стоимости проекта «Трейсер чистоты»

Дата оценки: 16.06.2026  
Валюта: казахстанский тенге (KZT)

## 1. Краткое описание проекта

«Трейсер чистоты» — веб-система для проведения внутренних аудитов/трейсеров в медицинской организации. Система позволяет аудиторам заполнять чек-листы, выбирать сотрудников и отделения, фиксировать результаты, прикладывать фото, автоматически рассчитывать процент соответствия и формировать отчеты.

## 2. Состав выполненных работ

В проект входят:

- frontend-приложение на Next.js / React / TypeScript;
- backend-приложение на Strapi v5 / TypeScript;
- модель данных: организации, программы, отделения, сотрудники, опросники, критерии, сессии трейсеров, результаты по субъектам;
- авторизация и разграничение сценариев для аудиторов/руководителей;
- формы проведения трейсеров с расчетом результата;
- поддержка разных типов опросников и шкал ответов;
- загрузка и хранение фото проверки;
- журнал проведенных трейсеров;
- аналитические отчеты по периодам, отделениям, сотрудникам, категориям и опросникам;
- графики, рейтинги проблемных вопросов, тепловая карта;
- экспорт отчетов в Excel и PDF;
- печатная форма одного трейсера;
- импорт/подготовка справочников сотрудников и опросников из Excel/DOCX;
- скрипты первичного наполнения базы;
- документация по локальному запуску и деплою.
